# LazyScripts
重复的事情就交给脚本去跑吧

## 目录
JavaInstaller--------------------------配置java环境变量

## 脚本用法
├── JavaInstaller
    ├── config_javac.sh
    ├── config_java.sh
    └── install_java.sh
`sudo JavaInstller/install_java.sh <jdk_folder>`
如果已经有其他版本的java，则需要执行`config_java.sh`和`config_javac.sh`两个脚本，选择当前版本即可
